# 発見ニュースまとめサイト

科学の新発見・考古学的発見・歴史的発見に関するニュースを、タイトル+概略+リンクの一覧としてまとめる静的サイト。ローカルニュース(日本語=日本、English=主要な英語圏国、Español=主要なスペイン語圏国)も扱う。GitHub Pagesで公開し、スマホからいつでも見られるようにする。

## カテゴリの方針

- 科学・考古学・歴史発見系: `考古学` / `古生物学` / `宇宙科学` / `言語学` / `生成AI` など、内容に応じて自由に設定してよい。
- ローカルニュース: 日本語タブは `ローカルニュース`(対象は日本)。English タブは主要な英語圏国(例: `Local News (US)` `Local News (UK)`)、Español タブは主要なスペイン語圏国(例: `Noticias Locales (España)` `Noticias Locales (Argentina)` `Noticias Locales (México)` `Noticias Locales (Perú)` `Noticias Locales (Chile)`)のように、カテゴリ名に国名を含めて区別する。

## 構成

- `index.html` : サイト本体。`data/news.json` を読み込んで一覧表示する。上部はスクロールしても追従するヘッダー(言語タブ/カテゴリフィルタ/「♥ お気に入りのみ」「未読のみ」トグル)。各記事のタグはタップでフィルタになり(同じタグを再タップで解除)、フィルタ中はヘッダー下に解除用チップが表示される。「すべて」表示時は記事がカテゴリ(ジャンル)ごとに`<details>`で折りたためるグループとして表示される(特定カテゴリ・タグ絞り込み中は通常のリスト表示)。PC幅ではカードが自動で複数列グリッド表示になる。
- `data/news.json` : 記事データ本体。`{ date, lang, category, title, summary, tags, image, url }` の配列。`lang` は `ja` / `en` / `es` のいずれか。`summary` は1〜2文程度の概略。`tags` は内容に関連するタグの配列。`image` はサムネイル画像URL(**任意項目**)。取得できた場合のみタイトル下に表示され、**未設定または読み込み失敗時はサムネイル欄ごと非表示**になる(プレースホルダーは表示しない)。

## いいね機能について

各記事のハートボタンでいいねを付けられる。**いいねの記録はスマホのブラウザのlocalStorageに保存されるだけで、GitHubリポジトリには保存されない**(静的サイトのため、サーバー側に書き戻す仕組みは持たせていない)。

次回の更新時に好みを反映してほしい場合は、サイト上部の「♥ 好みをコピー」ボタンを押すといいね済み記事のJSONがクリップボードにコピーされるので、それをClaude Codeとの会話に貼り付けて「これが好みだった」と伝える運用。

## 未読/既読機能について

記事のリンクをタップして開くと、自動的に「既読」として記録され(その記事は薄く表示される)、「未読のみ」トグルで未読分だけに絞り込める。いいねと同様、**この記録もスマホのブラウザのlocalStorageのみ**に保存される。

## 更新方法

家にいるときにClaude Codeへ「ニュースサイト更新して」のように依頼する(前回いいねした記事のJSONを貼り付けてもらえると、傾向を次回の記事選定に反映できる)。Claudeが以下を行う想定:

1. (あれば)貼り付けられた「いいね」記事の傾向を参考にする
2. Web検索で科学・考古学・歴史発見系、および各言語のローカルニュースを探す(日本語・英語・スペイン語それぞれ)
3. `data/news.json` に新しいエントリを追加。各エントリに1〜2文の`summary`、内容に沿った`tags`をClaudeが検索結果をもとに書く(`date` はYYYY-MM-DD形式、`lang` は記事の言語)。`image` はWebFetchで記事のメイン画像URLを取得できた場合のみ設定する(ドメインが記事と無関係に見える場合や取得失敗時は省略し、無理に埋めない)
4. `git add` → `git commit` → `git push`
5. pushするとGitHub Pagesが自動で再ビルドし、数分でサイトに反映される

## ローカルプレビュー

```
cd news-digest
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。
